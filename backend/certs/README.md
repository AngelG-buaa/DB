# SSL证书配置说明

## 证书文件放置

请将从TaurusDB下载的SSL证书文件放置在此目录下：

1. **CA证书文件**: `ca-bundle.pem` (必需)
   - 从TaurusDB实例管理页面下载
   - 用于验证服务器证书

2. **客户端证书文件** (可选):
   - `client-cert.pem`: 客户端证书
   - `client-key.pem`: 客户端私钥

## 下载步骤

1. 登录TaurusDB控制台
2. 进入"实例管理"页面
3. 点击实例名称进入实例概览页面
4. 点击"SSL"开关右侧的"下载"
5. 将下载的证书文件重命名为 `ca-bundle.pem` 并放置在此目录

## 配置说明

- 如果不使用SSL连接，请在 `.env` 文件中设置 `DB_SSL_ENABLED=false`
- 证书路径在 `.env` 文件中配置，支持相对路径和绝对路径
- 默认配置会验证服务器证书，如需跳过验证请设置 `DB_SSL_REJECT_UNAUTHORIZED=false`

## 安全提醒

- 请勿将证书文件提交到版本控制系统
- 确保证书文件权限设置正确
- 定期更新证书文件
